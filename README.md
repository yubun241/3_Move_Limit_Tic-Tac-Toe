## 3-Move Limit Tic-Tac-Toe
3つまでしか印を置けない、戦略的◯×ゲーム（三目並べ）です。

## 概要
Pythonista 3（iOS）で動作する、特殊ルールの◯×ゲームです。

通常の三目並べとは異なり、自分の印が盤面に最大3つまでしか存在できないため、引き分けがなく、常に盤面が変化し続ける戦略的なゲームプレイが楽しめます。

## 特殊ルール：4手目の制約
プレイヤーが4手目を打つと、そのプレイヤーが1番目に打った印（最も古い印）が自動的に消去されます。

盤面には常に各プレイヤーの最新の3手のみが残ります。

このルールにより、通常の三目並べで起こりやすい「引き分け」を回避し、先を読んだ配置が重要になります。


## ファイル構造
プロジェクトは以下の構造で管理されています。

01game/

├── src/

│   └── main.py    # ゲーム本体のソースコード

└── data/

    └── game_log.csv  # プレイ履歴が自動保存されるログファイル

## ログ記録機能
対局中のすべての手は data/game_log_YYYYMMDDhhmmss.csv に自動的に保存されます。

保存形式：player, hand, timestamp, x, y

player: 先行(0) / 後攻(1)

hand: 使用された印（◯ / ✖）

timestamp: 打った日時（YYYYMMDD_HHMMSS）

x, y: 盤面上の座標（0〜2）

## 動作環境
iOS / iPadOS

Pythonista 3

## 使い方
Pythonista 3 内に 01game フォルダを作成します。

src/main.py にコードを貼り付けて実行してください。

画面上のマスをタップして対局を開始します。
